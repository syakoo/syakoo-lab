このサイトを作り直し、本日2023年11月18日にバージョン6としてリリースしました 🎉

今までの振り返りと今後の方針について説明していきます。

## v6 をリリース

<Image
  src={"/img/writings/20231118/histories.png"}
  caption="このサイトの歴史"
  style={{ width: 500 }}
/>

このサイトは 2020 年、大学院生の時に v1 をリリースしました。
作成したモチベーションはイマイチ覚えてはいないですが、おそらく中学生くらいから密かに憧れていた研究所っぽいのを作りたかったとか、
単純に技術の勉強のためだったかなと思います。

当時は Firebase でコンテンツを保管していて Zeit (今の Vercel) にあげていたらしいです。
Zeit とかかなり懐かしく感じますね。

それから紆余曲折あって今のデザインっぽくなったのが v4 であり、この辺から大学院生を卒業し働き出しました。
こうしてみると個人開発にしては結構頑張ったなあと我ながらちょっと感心しています。

v5 から v6 で変わった点は、ざっと挙げると

- Vercel -> Cloudflare に変更
- vanilla-extract を採用

くらいとなります。

## 今後の方針

さて、新たなバージョンをリリースしたところでこのサイトの今後の方針について話します。

実は働き出してから仕事以外で活動する時間をあまり取ることができず、記事を全然あげなくなってしまっていたのですが、
最近仕事に慣れてきたのもありアウトプットが大事なことを再認識したのもありでこのサイトを頑張っていく気持ちになっています。

ただ、それでも記事という形でアウトプットしていくのは時間がかかり大変なので今後はアウトプットを目的としたゆるーーい書き物を Note として残していきたいと考えています。

今後もこのサイトは割と自己満で自分の都合の良いように更新されていくと思いますがよろしくお願いします。
