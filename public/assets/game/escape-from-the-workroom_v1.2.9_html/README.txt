# 作業部屋からの脱出 (v1.2.9)

ある日、フユがいつものようにハルの部屋で家事をしていたら、突然閉じ込められてしまった。
部屋に散らばる手がかりを探し、謎を解いて脱出することができるのか？

想定プレイ時間 20 分の短編脱出ゲームです

## はじめ方

**▼ ダウンロードして遊ぶ場合**
ダウンロードしたフォルダにある `index.html` ファイルを、お使いのウェブブラウザで開くとゲームが始まります。
一度ダウンロードすれば、インターネット環境のない場所（機内モードなど）でも、ブラウザさえあればいつでも遊ぶことができます。

※注意：
お使いのブラウザによっては、セキュリティの都合で正しく動作しない場合があります。
その際は、お手数ですがオンライン版でお楽しみください。

## 操作方法

### PC

- ↑↓←→, WASD ... 移動
- Space, Enter ... 決定
- Z ... メニュー表示切替

### スマートフォン

- 方向パッド ... 移動
- A ... 決定
- Z ... メニュー表示切替
